🔧 Configurações MikroTik

IP do roteador MikroTik na rede local
Padrão: 8728 (sem SSL) ou 8729 (SSL)
Nome configurado em IP → Hotspot → Servers

📋 Instruções de Configuração

1. Ativar API no MikroTik:
Vá em IP → Services e certifique-se que o serviço api está ativo na porta 8728.

2. Criar usuário API:
Em System → Users, crie um usuário com permissão write para o hotspot.

3. Configurar Hotspot:
Em IP → Hotspot, crie um servidor hotspot na interface desejada.

4. Redirecionar para este portal:
Em IP → Hotspot → Server Profiles, configure o campo Login Page para:
http://[IP_DESTE_SERVIDOR]:3000/?mac=$(mac)&ip=$(ip)&dst=$(link-orig)

💬 Configurações Zenvia SMS

Encontre em app.zenvia.com → API → Tokens
Nome que aparece no SMS (máx. 11 chars, sem espaços)

ℹ️ Sobre a Zenvia

A Zenvia é a plataforma de comunicação utilizada para enviar os códigos de verificação via SMS.

Para obter o token de API, acesse app.zenvia.com, faça login e vá em Configurações → API → Tokens.

O canal SMS deve estar habilitado na sua conta Zenvia.

Formato do SMS enviado:
"MG-NET SAL: Seu codigo de acesso e 123456. Valido por 5 minutos."
